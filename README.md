DESCRIPTION:

This project demonstrates a structured ETL (Extract, Transform, Load) pipeline for preprocessing raw data using Python and Scikit-learn. It begins by loading a dataset and inspecting the initial records to understand its structure. The code then automatically identifies numerical and categorical columns based on their data types. This separation allows for targeted preprocessing, ensuring that each type of data is handled appropriately without manual intervention. The goal is to clean and prepare the dataset in a systematic and reusable way.

The transformation phase uses pipelines to handle missing values and standardize the data. Numerical features are processed by filling missing values with the mean and scaling them, while categorical features are cleaned by replacing missing entries with a constant value such as "Unknown." These pipelines are combined using a column transformer, making the preprocessing efficient and consistent. The processed data is then converted back into a structured DataFrame, aligned with the original column order, and saved as a clean CSV file. This end-to-end pipeline ensures the dataset is ready for further analysis or machine learning tasks while maintaining clarity and reproducibility.
